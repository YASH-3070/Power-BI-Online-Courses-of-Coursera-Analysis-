# Power-BI-Online-Courses-of-Coursera-Analysis-
This dashboard provides an analytical overview of online courses across multiple categories such as Business, Computer Science, Data Science, Health, Personal Development, and more. It highlights key insights about course types, viewer engagement, language distribution, and ratings.

**Key Features:**
**Course Type Distribution:** Shows the count of Courses vs Specializations offered across all categories.

**Viewer Engagement:** Displays the average number of viewers by sub-category (e.g., History, Music & Art, Philosophy) and by number of subtitle languages.

**Language Insights:** A donut chart summarizing the distribution of courses by language (English in this dataset).

**Platform Summary (Coursera):** Highlights the average course rating and average number of viewers.

**Duration vs Viewership Table:** Compares course duration with the corresponding average number of viewers to identify trends in learner preferences.

The dashboard helps understand course popularity, user engagement patterns, and platform-level performance to guide better decision-making for content creators, educators, and analysts.
